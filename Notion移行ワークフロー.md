**目的**: Googleスプレッドシートで蓄積した問題をNotionに移行する手順

## 全体の流れ

1. Googleスプレッドシートに問題が溜まる
2. Claude Codeに「Notionに移行して」と伝える
3. 重複チェック → 自動で投入

## ⚠️ トラブルシューティング：スプレッドシートにアクセスできない場合

### Google Sheets の認証切れ・アクセスエラー

WebFetchで取得する際、**2ステップのリダイレクト**が必要。

**手順（パラセクトが実行すること）**:

1. まず元のエクスポートURLにWebFetchを投げる
   ```
   URL: https://docs.google.com/spreadsheets/d/1DQow9JiH1w9iO_GoD21mJZ6EXcPmrgc12NhpZaPQyWA/export?format=csv&gid=308773767
   ```
2. `307 Temporary Redirect` が返ってくるのでリダイレクト先URLを取得する
3. そのリダイレクト先URLに**すぐに**再度WebFetchを投げる
4. `400エラー`が出た場合は**Step1からやり直す**（リダイレクトURLは一時的なものなので期限切れになる）

**それでもアクセスできない場合（かずさんに依頼すること）**:

> 「スプレッドシートのCSVが取得できません。以下をターミナルに貼り付けて実行してもらえますか？」
> ```
> ! curl -L -o ~/Desktop/quiz.csv "https://docs.google.com/spreadsheets/d/1DQow9JiH1w9iO_GoD21mJZ6EXcPmrgc12NhpZaPQyWA/export?format=csv&gid=308773767"
> ```
> ※ `!` をつけてチャット入力欄から実行するとClaude Codeのセッションに結果が渡されます。

**補足**: Notionの認証切れで `/mcp` が必要な場合も同様に一度認証を依頼する。

## 移行ルール

### ✅ やること
- 問題文の前置き（「‼️かずからの挑戦状‼ 💫今日の問題です！！💫」など）を除去
- 問題文に選択肢を含める（1.〜 形式で改行）
- 解説は **`1.選択肢 → ○/× 理由`** 形式で整形して登録（2026-03-20以降ルール）
- 日付はスプレッドシートの「日にち」列の日付を使用
- **⚠️ 正解フォーマットは必ず数字のみ**
  - 単一選択: `2`
  - 複数選択（2つ選べ）: `2, 3`（カンマ区切りの数字のみ・テキスト不要）
  - ❌NG: `2. 水分出納の把握<br>3. 腸蠕動音の聴診` → アプリで複数選択として認識されない！

> 💡 解説フォーマット（完全版・必ず守ること）:
> ```
> 1.選択肢テキスト → ○/× 理由
> 2.選択肢テキスト → ○/× 理由
> 3.選択肢テキスト → ○/× 理由
> 4.選択肢テキスト → ○/× 理由
>
> 💡 かず学長のメモ: まとめ知識・学生コメントを活かした補足ポイント
> ```
> - 各行は `<br>` で区切る（2000文字以内）
> - `💡 かず学長のメモ:` は**必須**。スプレッドシートの学生コメントを活かして記載する
> - スプレッドシートの解説をそのまま使わず、**必ず Claude がこの形式に整形してから**投入する

### ❌ やってはいけないこと
- 重複チェックなしに投入しない
- 解説を省略しない（各選択肢に理由を書く）

### 重複チェック方法
1. `mcp__claude_ai_Notion__notion-search` で問題の核となるキーワードを検索
2. 該当カテゴリーの `data_source_url` を指定して検索
3. ヒットしたら → スキップ
4. ヒットなし → 投入OK

## Notion内の重複対処

**Notion API で直接削除可能（手動不要）**:
```python
requests.patch(f"https://api.notion.com/v1/pages/{page_id}",
    headers=HEADERS, json={"archived": True})
```
- 解説に `【重複・削除してください】` と書いてある問題を検出 → `archived: true` で削除
